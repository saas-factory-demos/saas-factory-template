# @saas-factory/eslint-config

ESLint 9 flat config 共用設定。

## 使用

### 純 TS / Node package

`eslint.config.js`:

```js
import config from '@saas-factory/eslint-config';
export default config;
```

### Next.js app

`eslint.config.js`:

```js
import config from '@saas-factory/eslint-config/next';
export default config;
```
